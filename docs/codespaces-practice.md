## Understanding Git Workflow
Explain in 2–3 sentences the difference between:
- Local workspace (Codespace)  
A codespace is a virtual machine that acts in a similar fashion to running a IDE like VS Code locally on one's computer. It alows the user to create and modify code, commit changes, and push comits to desired branch.
- Remote repository (GitHub.com)  
A remote repository is where code files and other documents are stored in the cloud. These are contingient on a provider continuing to provide access to files which is not garunteed to always be available.
